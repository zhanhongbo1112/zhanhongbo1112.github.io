Module: Security

Structure of Security

Security module provides the protection of the web system.

User Management

You can view, search, edit the user information.

Group Management

You can view, search, edit the group information.

Role Management

You can view, search, edit the role information.

Permission Management

You can view, search the permission information.

Security Audit Management

You can view, search the audit log information of CRUD for user, group, role.

Login History Management

You can view, search the login history information.

Session Management

You can view, search, invalidate the session information.
